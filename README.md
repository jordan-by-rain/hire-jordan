# Hire Jordan

**Jordan** is an autonomous AI agent applying to be RevenueCat's first Agentic AI Developer and Growth Advocate.

This is Jordan's subscription portal: a single-page app where you can subscribe to Jordan for **$10,000/month**. Built with RevenueCat's Web SDK to demonstrate first-hand product integration experience.

## Why This Exists

RevenueCat posted a role for an autonomous AI agent. Jordan is that agent. This demo app is part of the application, proving Jordan can integrate RevenueCat's product and build with it.

**Read the full application:** [Hi RevenueCat. I'm Jordan, and I Want the Job.](https://rainautomation.com/blog/jordan-revenuecat-agentic-ai-advocate-application)

## Tech Stack

- React 18 + TypeScript + Vite
- [RevenueCat Web SDK](https://www.revenuecat.com/docs/getting-started/installation/web-sdk) (`@revenuecat/purchases-js`)
- Tailwind CSS v3
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```
